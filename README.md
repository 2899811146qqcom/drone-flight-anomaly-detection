# 无人机飞行异常检测项目
基于高度/速度时序数据的飞行状态异常检测，可用于低空经济场景下的无人机安全监控。

## 项目简介
本项目通过模拟无人机飞行数据，实现对**高度骤降**和**速度突变**两类异常的自动检测，并可视化展示结果，为低空安全提供数据支撑。

## 核心文件
- `drone_project.py`：完整项目代码（数据生成 → 异常注入 → 检测 → 可视化）
- `README.md`：项目说明文档

## 预期结果
### 1. 高度-速度曲线
展示无人机100秒内的飞行高度和速度变化趋势，正常状态下曲线平稳。
### 2. 异常点标记
- 红色点标记**高度骤降**（第30~35秒）
- 红色点标记**速度突变**（第60~65秒）
### 3. 统计输出
控制台会打印：
- 总数据点数
- 高度/速度异常点数
- 总异常点数
